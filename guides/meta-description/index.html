A meta description for every page

Worth +8 points · the summary crawlers and previews show

The meta description is a one-sentence summary of a page. Search engines, AI answer engines and link-preview cards often show it verbatim, so it's your chance to say — in plain language — what the page offers and why someone should open it.

Copy-paste: where it goes

Inside <head>, one per page:

<meta name="description"
      content="Grade any website for AI-readiness in seconds — check llms.txt,
               robots.txt AI-bot policy, structured data and more. Free.">

What makes a good one

AvoidPrefer
Empty / missingA clear one-sentence summary
Keyword list with no sentenceNatural language a person would read
Same description site-wideA description specific to each page

Rules of thumb

Aim for roughly 120–160 characters — long enough to be useful, short enough not to be cut off. Lead with the concrete benefit. Write it for a human; if it reads naturally, crawlers handle it fine. Give each page its own.
